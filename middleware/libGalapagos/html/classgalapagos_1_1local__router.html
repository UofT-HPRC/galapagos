Detailed Description

template<class T>
class galapagos::local_router< T >

Class for the local_router.

A software implementation of the node router, routing packets by dest sidechannel. Routes between input interfaces(s_axis) and an output interface(m_axis).

void galapagos::local_router< T >::route ( )

routing function, can be overridden for different routers

Routing function Round robins through the m_axis (the outputs of all kernels and external drivers). Reads header_dest of interface and splices into correct s_axis based off dest. This keeps running until the done is asserted externally.
