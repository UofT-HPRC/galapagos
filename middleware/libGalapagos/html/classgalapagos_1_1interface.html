Detailed Description

template<class T>
class galapagos::interface< T >

Class for the Galapagos Interface.

Threadsafe for single producer, single consumer. Gives user an interface to read/write flits and entire packets. Packets internally are stored in linked-list and can be spliced to other linked-lists. The flit interface reads from the head packet of the linked-list and writes to the tail packet.
